# Emlak CRM (Mini)

Emlak ofisleri için **tanıtım sitesi + giriş sonrası CRM paneli** mantığında hazırlanmış mini bir proje.  
Backend: **NestJS + Prisma + PostgreSQL + JWT**  
Frontend: **React + Vite**

---

## Özellikler

### Web Site (Public)
- Ana sayfa / Hakkımızda / İletişim sayfaları
- Mobil uyumlu tasarım
- Dark / Light tema

### CRM Panel (Login sonrası)
- İlan yönetimi (CRUD): ekle / düzenle / sil
- Filtreleme: tür, durum, fiyat aralığı + arama
- Sayfalama (frontend)
- JWT token ile korumalı sayfalar
